Q: What is Java?

Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle).

Key Points:
  • Platform Independent: Write Once, Run Anywhere (WORA)
  • Object-Oriented: Everything is an object
  • Simple: Easy to learn (similar to C/C++)
  • Secure: No pointer, bytecode verification
  • Multithreaded: Multiple tasks simultaneously
// Simple Java Program
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java World!");
    }
}

Why Learn Java?

  • Android App Development
  • Web Applications (Spring Boot)
  • Enterprise Software
  • Big Data & Machine Learning
Q: What are the main features of Java?

Java has several powerful features:

Main Features:
  1. Simple: Easy syntax, no complex pointers
  2. Object-Oriented: Follows OOP principles
  3. Platform Independent: Bytecode runs on any JVM
  4. Robust: Automatic garbage collection
  5. Secure: No explicit pointer, bytecode verification
  6. Multithreading: Concurrent execution
  7. Distributed: Network programming support

Example of Platform Independence:

// This same code runs on Windows, Linux, Mac
public class PlatformDemo {
    public static void main(String[] args) {
        String os = System.getProperty("os.name");
        System.out.println("Running on: " + os);
    }
}
Q: What is JVM, JRE and JDK?

These are three core components of Java platform:

Component Full Form Purpose
JDK Java Development Kit For developers (JRE + development tools)
JRE Java Runtime Environment For running Java programs (JVM + libraries)
JVM Java Virtual Machine Executes bytecode (platform dependent)
Relationship:

JDK contains JRE contains JVM

Q: How to write first Java program?

Follow these simple steps:

Steps:
  1. Install JDK on your computer
  2. Write code in a text editor
  3. Save file with .java extension
  4. Compile using javac command
  5. Run using java command
// File: Hello.java
public class Hello {
    public static void main(String[] args) {
        System.out.println("My First Java Program!");
        System.out.println("Java is fun to learn!");
    }
}

Compilation & Execution:

// Open terminal/command prompt
javac Hello.java   // Compiles the code
java Hello         // Runs the program

// Output:
// My First Java Program!
// Java is fun to learn!

Q: What are Operators in Java?

Operators are used to perform operations on variables and values.

// Arithmetic Operators
int a = 10, b = 5;
int sum = a + b;      // 15
int diff = a - b;     // 5
int product = a * b;  // 50
